# THE WATCHER PROTOCOL

> *"No loud noises. No cheap jump scares. Just the slow realization that something is fundamentally wrong."*

A psychological horror mod for Minecraft that creates genuine dread through environmental manipulation, behavioral learning, and subtle reality corruption. This is not your typical horror mod - it's designed to unsettle, not startle.

---

## üé≠ Philosophy

THE WATCHER PROTOCOL rejects conventional horror tropes:

- ‚ùå No loud noises
- ‚ùå No predictable jump scares  
- ‚ùå No obvious monsters stalking you every five minutes

Instead, it provides:

- ‚úÖ **Uncertainty** - Nothing behaves quite right
- ‚úÖ **False Safety** - Your base isn't safe anymore
- ‚úÖ **Subtle Manipulation** - Small changes that accumulate
- ‚úÖ **Delayed Consequences** - Your actions unlock your doom
- ‚úÖ **Personal Horror** - The mod learns and imitates your behavior

**Fear comes from loss of trust in reality, not noise.**

## üåü Key Features

### Behavioral Learning System
The mod tracks everything you do:
- Mining patterns and preferred depths
- Base layouts and building styles
- Movement routes and habits
- Sleep cycles and timing
- Chest organization patterns

This data is used to create **personal, targeted horror** that feels unique to your playstyle.

### Seven Horror Phases

#### Phase 1: The Unsettling
- Crops grow irregularly
- Furnace times fluctuate randomly
- Redstone occasionally double-ticks
- Mob AI shows subtle anomalies

#### Phase 2: The Doubt
- Distant footsteps with no source
- False mining sounds underground
- Door sounds without doors moving
- Torches appear where you didn't place them

#### Phase 3: The Presence
- Shadow figures in the fog
- Impossible water reflections
- Subtle camera micro-tilts
- Breathing sounds when at full health

#### Phase 4: The Interference
- Doors open slightly at night
- Crops uproot themselves
- Tunnels form behind you
- Biomes bleed into each other

#### Phase 5: The Mockery
- Footsteps matching your exact walking pattern
- Mining sounds mimicking your rhythm
- Underground duplicates of your base (with something wrong)
- Your patterns, copied and distorted

#### Phase 6: The Confrontation
- **The Watcher manifests**
- 3 blocks tall, silently watching
- Moves at 90% normal speed
- Disappears when stared at too long
- Slowly approaches if ignored

#### Phase 7: The Unraveling
- Chunk borders become visible
- Terrain loads incorrectly
- Physics occasionally fails
- Impossible geometry appears
- Reality itself breaks down

### No Victory Condition
There is no boss to defeat. No final battle. The only way to stop it:

**Abandon the world.**

But if you regenerate the same seed... **Phase 3 begins immediately.** The world remembers.

## üì¶ Installation

### Requirements
- Minecraft 1.20.1
- Fabric Loader 0.15.0+
- Fabric API
- Cardinal Components API
- (Optional) Cloth Config API for in-game configuration

### Steps
1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Download Fabric API from [Modrinth](https://modrinth.com/mod/fabric-api) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
3. Download THE WATCHER PROTOCOL
4. Place all files in your `mods` folder
5. Launch Minecraft

## üéÆ Gameplay Tips

### Surviving the Protocol

1. **Pay Attention to Details**
   - Document what seems wrong
   - Keep notes on patterns
   - Trust your instincts

2. **Don't Ignore the Signs**
   - Early warning signs matter
   - Small changes compound
   - Phases activate based on your behavior

3. **Understand the Triggers**
   - Mining unlocks progression
   - Building a base increases intensity
   - Time in-world accelerates phases
   - Death has consequences

4. **Know When to Leave**
   - Some worlds cannot be saved
   - Starting fresh may be the only option
   - But the seed remembers...

### What NOT to Do

- Don't dismiss anomalies as bugs
- Don't build elaborate redstone early
- Don't ignore the audio cues
- Don't think your base is safe
- Don't try to fight The Watcher

## ‚ùì FAQ

**Q: Is this compatible with other mods?**
A: Yes, we use minimal mixins and event-based integration for maximum compatibility.

**Q: Can I disable specific phases?**
A: Yes, every phase can be individually toggled in the config.

**Q: Does this work on servers?**
A: Single-player focused, but basic multiplayer support exists. Full MP features coming in v1.1.

**Q: How do I uninstall completely?**
A: Delete the mod file and the `config/watcherprotocol` folder. World data persists in save files.

**Q: Will this lag my game?**
A: Performance optimizations are built-in. Use performance mode if needed.

**Q: Is there a way to "win"?**
A: No. Abandoning the world is the only escape.

**Q: What happens if I regenerate an infected seed?**
A: Phase 3 activates immediately. The world never forgets.
